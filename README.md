# Wheel-of-Fortune
This is a wheel of fortune written in C++ on WinForms. Supports one user at a time. The administrator enters questions, and the Player can spin the wheel, answer questions as correct-wrong, collect or lose points for answers, and rise or fall in the ranking table.
